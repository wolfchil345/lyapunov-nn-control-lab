# 方法論

このドキュメントでは、Lyapunov NN Control Labで使用する主要な制御工学の考え方を説明します。

## 1. 質量ばねダンパ系

このプロジェクトでは、2次の質量ばねダンパ系を対象とします。

```text
x = [位置, 速度]
```

システムは状態空間形式で表されます。

```text
dx/dt = A x + B u
```

ここで、xは状態、uは制御入力、Aは制御対象の動特性、Bは入力が制御対象へ与える影響を表します。

## 2. LQR基準制御器

線形二次レギュレータを、古典的な最適制御の基準として使用します。

```text
u = -Kx
```

LQRは、ニューラルネットワークが模倣するための強力な基準制御器です。

## 3. ニューラルネットワーク制御器

ニューラルネットワーク制御器は、位置と速度を入力として受け取り、1つのスカラー制御入力を出力します。

```text
NN(x) ≈ LQR(x)
```

これは模倣学習であり、ニューラルネットワークはサンプリングされた状態上でLQR制御器のふるまいを学習します。

## 4. Lyapunov安定性確認

Lyapunov関数は、安定性を考察するために使用されるエネルギーのような関数です。

```text
V(x) = x^T P x
```

安定な閉ループ系では、軌道に沿ってLyapunov関数の値が減少する必要があります。

```text
dV/dt < 0
```

このプロジェクトでは、サンプリングした状態のグリッド上でこの条件を確認します。

## 5. 安定性を考慮した学習

ニューラルネットワークは、模倣損失とLyapunov安定性ペナルティの両方を用いて学習します。

```text
総損失 = 模倣損失 + 安定性ペナルティ
```

安定性ペナルティは、Lyapunov関数の時間微分を正にする制御出力を抑制します。

## 6. アクチュエータ飽和

実際のアクチュエータは無制限の力を加えることができないため、飽和制御も評価します。

```text
u = clip(u, -u_max, u_max)
```

これにより、アクチュエータの制限が閉ループ安定性と制御性能に与える影響を確認します。

## 7. ロバスト性実験

学習した制御器が、不完全な条件でも有効に機能するかを評価します。

ノイズロバスト性実験では、計測値にノイズを加えます。

```text
x_measured = x + noise
```

パラメータロバスト性実験では、質量、減衰係数、ばね定数を変更し、モデル化誤差を再現します。

## 8. 位相平面図とLyapunov等高線

位相平面図では、位置と速度をプロットし、軌道が原点へ向かうかを確認します。

Lyapunov等高線図では、Lyapunov関数の等位集合上に軌道を重ねて表示します。

これらの図により、閉ループ系の安定挙動を視覚的に説明できます。

## 9. 吸引領域解析

吸引領域実験では、多数の初期状態をシミュレーションし、それぞれを収束または非収束として分類します。

この実験は、次の問いに答えます。

```text
どの初期状態から制御器はシステムを安定化できるか？
```

LQR、ニューラルネットワーク制御、飽和ニューラルネットワーク制御の吸引領域も比較します。

## 10. 安定性重みのアブレーション研究

アブレーション研究では、異なるLyapunovペナルティ重みを使用して複数の制御器を学習します。

安定性をより強く考慮した学習が、Lyapunov条件違反率、最終状態ノルム、整定時間、二次コスト、制御エネルギーなどを改善するかを評価します。

## 11. まとめ

このプロジェクトは、古典制御、ニューラルネットワークによる模倣学習、Lyapunov解析、ロバスト性試験、経験的な吸引領域推定を組み合わせています。

主要な研究課題は次のとおりです。

```text
ニューラルネットワーク制御器は、安定化可能な古典制御器を模倣しながら、Lyapunov安定性解析によって評価できるか？
```
