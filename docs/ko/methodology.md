# 방법론

이 문서는 Lyapunov NN Control Lab에서 사용하는 주요 제어공학 개념을 설명합니다.

## 1. 질량-스프링-댐퍼 시스템

이 프로젝트는 2차 질량-스프링-댐퍼 시스템을 대상으로 합니다.

```text
x = [위치, 속도]
```

시스템은 상태공간 형태로 표현됩니다.

```text
dx/dt = A x + B u
```

여기서 x는 상태, u는 제어 입력, A는 플랜트의 동역학, B는 입력이 플랜트에 미치는 영향을 나타냅니다.

## 2. LQR 기준 제어기

선형 이차 레귤레이터를 고전적인 최적제어 기준으로 사용합니다.

```text
u = -Kx
```

LQR은 신경망이 모방할 수 있는 강력한 기준 제어기를 제공합니다.

## 3. 신경망 제어기

신경망 제어기는 위치와 속도를 입력으로 받고 하나의 스칼라 제어 입력을 출력합니다.

```text
NN(x) ≈ LQR(x)
```

이 과정은 모방 학습이며, 신경망은 샘플링된 상태에서 LQR 제어기의 동작을 학습합니다.

## 4. Lyapunov 안정성 확인

Lyapunov 함수는 안정성을 분석하기 위해 사용하는 에너지와 유사한 함수입니다.

```text
V(x) = x^T P x
```

안정적인 폐루프 시스템에서는 궤적을 따라 Lyapunov 함수의 값이 감소해야 합니다.

```text
dV/dt < 0
```

이 프로젝트는 샘플링된 상태 격자에서 이 조건을 확인합니다.

## 5. 안정성을 고려한 학습

신경망은 모방 손실과 Lyapunov 안정성 페널티를 함께 사용하여 학습됩니다.

```text
전체 손실 = 모방 손실 + 안정성 페널티
```

안정성 페널티는 Lyapunov 미분값을 양수로 만드는 제어 출력을 억제합니다.

## 6. 액추에이터 포화

실제 액추에이터는 무한한 힘을 발생시킬 수 없으므로 포화된 제어 입력도 실험합니다.

```text
u = clip(u, -u_max, u_max)
```

이를 통해 액추에이터 제한이 폐루프 안정성과 성능에 미치는 영향을 확인합니다.

## 7. 강건성 실험

학습된 제어기가 불완전한 조건에서도 효과적으로 동작하는지 평가합니다.

노이즈 강건성 실험에서는 측정값에 노이즈를 추가합니다.

```text
x_measured = x + noise
```

파라미터 강건성 실험에서는 질량, 감쇠, 강성을 변경하여 모델링 오차를 모의합니다.

## 8. 위상 궤적과 Lyapunov 등고선

위상 궤적은 위치와 속도를 함께 표시하여 궤적이 원점으로 이동하는지 보여줍니다.

Lyapunov 등고선 그림은 Lyapunov 함수의 레벨 집합 위에 시스템 궤적을 표시합니다.

이 그림들은 폐루프 안정성 거동을 시각적으로 설명합니다.

## 9. 끌림 영역 분석

끌림 영역 실험에서는 여러 초기 상태를 시뮬레이션하고 각각을 수렴 또는 비수렴으로 분류합니다.

이 실험은 다음 질문에 답합니다.

```text
어떤 초기 상태에서 제어기가 시스템을 안정화할 수 있는가?
```

LQR, 신경망 제어, 포화 신경망 제어의 끌림 영역도 비교합니다.

## 10. 안정성 가중치 어블레이션 연구

어블레이션 연구에서는 서로 다른 Lyapunov 페널티 가중치로 제어기를 학습합니다.

더 강한 안정성 중심 학습이 Lyapunov 조건 위반 비율, 최종 상태 노름, 정착 시간, 이차 비용, 제어 에너지 등의 지표를 개선하는지 평가합니다.

## 11. 요약

이 프로젝트는 고전 제어, 신경망 모방 학습, Lyapunov 분석, 강건성 실험, 경험적 끌림 영역 추정을 결합합니다.

주요 연구 질문은 다음과 같습니다.

```text
신경망 제어기는 안정화 가능한 고전 제어기를 모방하면서 Lyapunov 기반 안정성 도구로 평가될 수 있는가?
```
