# 다국어화 가이드

이 프로젝트는 방문자가 읽는 문서를 네 가지 언어로 제공하는 것을 목표로 합니다.

- English
- 日本語
- 한국어
- ไทย

## 번역 정책

README, 문서, 릴리스 노트, 기여 가이드, Issue 템플릿, Pull Request 템플릿처럼 방문자가 읽는 내용을 번역합니다.

Python 함수 이름, 코드에서 사용하는 파일 이름, workflow 문법, 패키지 설정, 명령어 이름은 번역하지 않습니다.

## 폴더 구조

```text
docs/en/
docs/ja/
docs/ko/
docs/th/
```

번역된 문서는 모든 언어 폴더에서 같은 파일 이름을 사용합니다.

## 언어 전환 규칙

각 번역 문서의 맨 위에는 네 가지 언어로 이동할 수 있는 링크를 둡니다.

## 릴리스 규칙

주요 방문자용 문서가 네 가지 언어로 준비된 뒤 최종 포트폴리오 릴리스를 만듭니다.
