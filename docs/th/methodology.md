# ระเบียบวิธี

เอกสารนี้อธิบายแนวคิดหลักด้านวิศวกรรมควบคุมที่ใช้ใน Lyapunov NN Control Lab

## 1. ระบบมวล-สปริง-แดมเปอร์

โปรเจกต์นี้ศึกษาระบบมวล-สปริง-แดมเปอร์อันดับสอง

```text
x = [ตำแหน่ง, ความเร็ว]
```

ระบบเขียนในรูปแบบปริภูมิสถานะได้ดังนี้

```text
dx/dt = A x + B u
```

โดย x คือสถานะ u คืออินพุตควบคุม A อธิบายพลวัตของระบบ และ B อธิบายผลของอินพุตที่มีต่อระบบ

## 2. ตัวควบคุมพื้นฐาน LQR

ใช้ Linear Quadratic Regulator เป็นตัวควบคุมอ้างอิงของการควบคุมเหมาะที่สุดแบบดั้งเดิม

```text
u = -Kx
```

LQR เป็นตัวควบคุมอ้างอิงที่มีประสิทธิภาพสำหรับให้ neural network เรียนรู้และเลียนแบบ

## 3. ตัวควบคุมโครงข่ายประสาทเทียม

ตัวควบคุม neural network รับตำแหน่งและความเร็วเป็นอินพุต และส่งออกอินพุตควบคุมแบบสเกลาร์หนึ่งค่า

```text
NN(x) ≈ LQR(x)
```

กระบวนการนี้คือ imitation learning โดย neural network เรียนรู้พฤติกรรมของ LQR controller จากสถานะที่สุ่มตัวอย่าง

## 4. การตรวจสอบเสถียรภาพแบบ Lyapunov

ฟังก์ชัน Lyapunov เป็นฟังก์ชันที่คล้ายพลังงานและใช้ในการวิเคราะห์เสถียรภาพ

```text
V(x) = x^T P x
```

สำหรับระบบวงปิดที่เสถียร ค่าของฟังก์ชัน Lyapunov ควรลดลงตามวิถีของระบบ

```text
dV/dt < 0
```

โปรเจกต์ตรวจสอบเงื่อนไขนี้บนกริดของสถานะที่สุ่มตัวอย่าง

## 5. การฝึกที่คำนึงถึงเสถียรภาพ

neural network ถูกฝึกด้วยทั้ง imitation loss และ Lyapunov stability penalty

```text
total loss = imitation loss + stability penalty
```

stability penalty ช่วยลดการสร้างอินพุตควบคุมที่ทำให้อนุพันธ์ของฟังก์ชัน Lyapunov เป็นบวก

## 6. การอิ่มตัวของแอคชูเอเตอร์

แอคชูเอเตอร์จริงไม่สามารถสร้างแรงได้อย่างไม่จำกัด จึงมีการทดสอบอินพุตควบคุมแบบอิ่มตัวด้วย

```text
u = clip(u, -u_max, u_max)
```

การทดลองนี้แสดงผลของขีดจำกัดแอคชูเอเตอร์ต่อเสถียรภาพและสมรรถนะของระบบวงปิด

## 7. การทดลองความทนทาน

โปรเจกต์ประเมินว่าตัวควบคุมที่เรียนรู้แล้วยังคงทำงานได้ดีภายใต้สภาวะที่ไม่สมบูรณ์หรือไม่

การทดลองความทนทานต่อสัญญาณรบกวนจะเพิ่ม noise ให้กับค่าที่วัดได้

```text
x_measured = x + noise
```

การทดลองความทนทานต่อพารามิเตอร์จะเปลี่ยนมวล ค่าหน่วง และค่าความแข็งของสปริง เพื่อจำลองความคลาดเคลื่อนของแบบจำลอง

## 8. แผนภาพเฟสและเส้นชั้นของ Lyapunov

แผนภาพเฟสพล็อตตำแหน่งเทียบกับความเร็ว และแสดงว่าวิถีเคลื่อนเข้าสู่จุดกำเนิดหรือไม่

กราฟเส้นชั้น Lyapunov แสดงวิถีของระบบทับบนระดับค่าของฟังก์ชัน Lyapunov

กราฟเหล่านี้ช่วยอธิบายพฤติกรรมเสถียรภาพของระบบวงปิดด้วยภาพ

## 9. การวิเคราะห์บริเวณดึงดูด

การทดลองบริเวณดึงดูดจำลองสถานะเริ่มต้นหลายค่า และจำแนกแต่ละค่าเป็นลู่เข้าหรือไม่ลู่เข้า

การทดลองนี้ตอบคำถามต่อไปนี้

```text
ตัวควบคุมสามารถทำให้ระบบเสถียรได้จากสถานะเริ่มต้นใดบ้าง?
```

โปรเจกต์ยังเปรียบเทียบบริเวณดึงดูดของ LQR, neural-network control และ saturated neural-network control

## 10. การศึกษา ablation ของค่าน้ำหนักเสถียรภาพ

การศึกษา ablation ฝึกตัวควบคุมด้วยค่าน้ำหนัก Lyapunov penalty ที่แตกต่างกัน

การทดลองตรวจสอบว่าการฝึกที่เน้นเสถียรภาพมากขึ้นช่วยปรับปรุงสัดส่วนการละเมิดเงื่อนไข Lyapunov, final state norm, settling time, quadratic cost และ control energy หรือไม่

## 11. สรุป

โปรเจกต์นี้ผสมผสานการควบคุมแบบดั้งเดิม การเรียนรู้แบบเลียนแบบด้วย neural network การวิเคราะห์ Lyapunov การทดสอบความทนทาน และการประมาณบริเวณดึงดูดจากการทดลอง

คำถามวิจัยหลักคือ

```text
ตัวควบคุม neural network สามารถเลียนแบบตัวควบคุมแบบดั้งเดิมที่ทำให้ระบบเสถียร พร้อมทั้งได้รับการประเมินด้วยเครื่องมือวิเคราะห์เสถียรภาพแบบ Lyapunov ได้หรือไม่?
```
